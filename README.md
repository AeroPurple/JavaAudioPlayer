# φb's Audio Player
Java-based audio player with visualizations I initially made as a final project for one of my high school classes.
<img width="646" height="484" alt="Screenshot of the audio player, playing back stereo audio, with a horizontal oscilloscope visualizer" src="https://github.com/user-attachments/assets/c0392ea4-7df6-4f58-97c2-4bc690d48cea" />
### File formats supported
| File Type | Support |
| --- | --- |
| **WAV, 8-bit** | **✓** Yes |
| **WAV, 16-bit** | **✓** Yes |
| **WAV, 24-bit** | **✗** No |
| **WAV, 32-bit** | **✗** No |
| **MP3** | **✗** No |
| **AAC** | **✗** No |
| **M4A** | **✗** No |
